Groth16 Go WASM POC

This page compares an end-to-end Groth16 proof flow in the browser for fixed no-commitment circuit fixtures on BN254 or BLS12-381. One path uses the local WebGPU-accelerated prover package in this repo; the other uses native gnark Groth16 compiled to wasm. The benchmark loads a serialized circuit and proving/verification keys, builds the witness, proves, serializes the proof, deserializes it, and verifies it. Proof bytes are intentionally not compared because Groth16 proving is randomized.
